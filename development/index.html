# Introduction

The chora project is a research project. All tools and services being built within the chora project are prototypes and proof-of-concepts and they are all subject to change.

# Chora Chain

chora-chain (opens new window) is an experimental blockchain built with Cosmos SDK (opens new window) and Starport (opens new window).

# Chora Modules

chora-chain (opens new window) includes custom modules that can be used by other Cosmos-based blockchains.

# Ecodex Module

The ecodex module was originally built from the interchange exchange tutorial (opens new window) and is currently being modified to support the exchanging of tokens for "eco credits" (or more specifically vouchers for locked "eco credits") from regen-ledger (opens new window) through other Cosmos-based blockchains.
